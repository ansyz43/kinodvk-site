# ТЗ — Сайт «Дальневосточная Кинопремия»

## 1. Цели MVP
- **Промо-лендинг** в фирменном стиле (HTML/CSS — точная копия макета, см. [site/index.html](site/index.html)).
- **Кнопка «Оставить заявку»** → модалка (имя + телефон + согласие) → **письмо** на ящик получателя заявок.
- Запуск на **VPS Timeweb** под доменом `кинопремиядв.рф`.

## 2. Аудитория
Кинопрофессионалы, пресса, партнёры, широкая публика.

## 3. Дизайн
**Фиксированный CSS из брифа** — не переделываем:
- Цвета: `--amber #FF9500`, `--amber2 #FFD000`, `--red #E8341A`, `--deep #0A0A0F`, `--dark #0E0E16`, `--mid #161620`, `--card #13131C`, `--text #E8E4DC`, `--muted #5A5770`.
- Шрифты: `Bebas Neue`, `Cormorant Garamond`, `Space Mono` (Google Fonts, кириллица).
- Эффекты: noise-overlay, beacon, hero-лучи, shimmer-band — копия макета.

## 4. Структура
- Один HTML-файл-лендинг с разделами: Nav · Hero · Band · About · Nominations · Programme · Submit · Partners · News · Footer.
- Все CTA «Подать заявку» (`[data-lead-open]`) открывают одну модалку.

## 5. Функционал заявки
- Поля: **Имя** (2–80 симв.), **Телефон** (мин. 6 цифр, макс. 32 симв.), чекбокс согласия 152-ФЗ, скрытое honeypot-поле `company`.
- Клиент: валидация, автоформат телефона, статус `Отправляем / Отправлено / Ошибка`.
- Сервер `POST /api/lead` (Node) или `POST /mail.php` (Beget) → отправка письма через SMTP (`smtp.beget.com`) или `mail()` на `MAIL_TO` (по умолчанию `kisuke43@gmail.com`).
- Защита: rate-limit 5 req/min/IP, honeypot, длина полей, экранирование HTML в письме.

## 6. Сообщение в письме
```
Тема: Новая заявка — Кинопремия

🎬 Новая заявка — Кинопремия
Имя: <name>
Телефон: <phone>
Страница: /
Время (Влд): 04.05.2026, 21:50:00
IP: x.x.x.x
UA: ...
```

## 7. Нефункциональные
- Адаптивность от 360px (медиазапрос ≤900px уже в стилях).
- HTTPS обязательно (Let's Encrypt через Caddy/Nginx).
- Время ответа `/api/lead` ≤ 2 сек.
- Аптайм одного VPS достаточно, бэкап `.env` и `index.html` руками или через git.

## 8. Out of scope (можно добавить позже)
- CMS, архив фильмов с фильтрами, страница «О премии», «Новости» с админкой.
- Загрузка ссылки на скринер, выбор номинации в форме.
- Интеграция с moviestart.ru (внешняя ссылка как опция в модалке).
- Ya.Метрика, sitemap, OG-картинки.
